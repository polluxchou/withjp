# Chinese and English I18n Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the web app switch completely between Chinese and English for the main application surface.

**Architecture:** Use `next-intl` as the single translation system. Locale-aware routes live under `/[locale]`, shared navigation preserves the current path while switching languages, and message files are the only source of UI labels.

**Tech Stack:** Next.js App Router, `next-intl`, React Server Components, client components, TypeScript.
